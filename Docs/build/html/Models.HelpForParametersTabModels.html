Models.HelpForParametersTabModels.preferenceLearning module¶

class Models.HelpForParametersTabModels.preferenceLearning.PreferenceLearning(master, prometheeGamma: PrometheeGamma)[source]¶

Bases: object

A class to implement a preference learning algorithm

This algorithm is based on a small number of pairwise comparisons submitted to the user. Based on the user’s responses, an iterative search determines the possible values for the 3 parameters specific to the PROMETHEE Gamma method.
