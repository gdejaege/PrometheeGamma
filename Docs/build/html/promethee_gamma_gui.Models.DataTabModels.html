build_pi_c_matrix() → None[source]¶

Build the pi matrix and the phi list of the criterion

πc_ij = Fc(dc(ai, aj)): “how much ai is preferred over aj on criterion c”

Fc is the preference function and dc(ai, aj) = fc(ai) - fc(aj ) is the difference between the evaluations of ai and aj on the criterion c
